# Fragments

# Running the Project

This project includes several npm scripts to help with development.

Use These commands from the project root.

# Start the Server

npm start - This will execute node src/server.js

The server will start on http://localhost:8080

# Development Mode

npm run dev

This command use nodemon so the server reloads whenever you save a file.

# Debug Mode

npm run debug

This command start the server with Node' built-in debugger enabled.

# Linting the Code

npm run lint

This command will report any style or syntax issues based on my ESLint configuration.

To test the server in browser:

http://localhost:8080

Or Using curl

curl.exe localhost:8080
